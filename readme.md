### create venv
```shell
cd flask-vue-demo

python -m venv venv

[mac/linux]:
  source venv/bin/activate
  
[windows]:
  venv\Scripts\activate

```
### install requirements
```shell
pip install -r requirements.txt

```

### create log file
```shell
Create the Logs folder under the project directory and create the Log file
: /logs/log
```


### start project
```
mac/linux:
    start.sh
windows:
    start.bat
```
